# QQ 机器人

这是一个用于群聊的 QQ 机器人，当前支持：

- Dota2 战绩播报、手动查询和专业问答
- 联网问答与今日新闻
- 带上下文的群聊互动

## 使用方式

在群里 `@机器人` 后使用。

### 指令入口

- `/help`
  - 查看当前功能简介
- `/list`
  - 查看当前群监听的 Dota2 昵称
- `/add <昵称> <steamID>`
  - 给当前群新增 Dota2 监听账号
- `/push <昵称>`
  - 查看该昵称最近一场比赛的战绩点评
- `/dota_analyze <昵称或steamID>`
  - 统计该账号最近 50 场的胜率、常用英雄和极值战绩
- `/dota_profile <昵称或steamID>`
  - 基于最近比赛做深度打法画像和改进建议
- `/dota_guide <英雄名>`
  - 输出当前版本的深度英雄攻略
- `/news`
  - 查看当天热点新闻
- `/news <关键词>`
  - 查看当天指定主题新闻
- `/todo`
  - 查看当前待办
- `/todo <文本内容>`
  - 提交一个新的功能建议

未识别的 `/` 指令会直接提示“未识别的指令”。

### 自然语言入口

除了命令，也可以直接和机器人说话：

- 问功能
  - 例如：`你有什么功能`
- 问 Dota2
  - 例如：`蓝猫怎么出装`、`这版本三号位谁强`
- 问实时信息
  - 例如：`今天北京天气怎么样`
- 普通闲聊或吐槽机器人
  - 机器人会参考最近群聊上下文来回复

### 群聊互动

机器人会结合最近两小时的群聊内容来接话。

- 普通闲聊会自然回复
- 吐槽或批评机器人时，会接梗并简短回怼
- 如果群里冷场太久，机器人可能会主动讲一个很冷的笑话
