# QQ 机器人

这是一个面向群聊使用的 QQ 机器人，当前主要提供两类能力：

- Dota2 战绩播报
- 群内问答与轻互动

## 可以怎么用

在群里 `@机器人` 后使用。

### 指令入口

- `/help`
  - 查看机器人当前能做什么
- `/list`
  - 查看当前群正在监听的账号昵称
- `/add <昵称> <steamID>`
  - 给当前群新增一个 Dota2 监听账号
- `/push <昵称>`
  - 拉取该昵称对应账号的最新一场比赛，并在当前群生成一条战绩点评
- `/todo`
  - 查看当前待办功能
- `/todo <文本内容>`
  - 提交一个新的功能建议

如果输入的是未知 `/` 指令，机器人会直接提示“未识别的指令”。

### 自然语言入口

除了指令，也可以直接用自然语言和机器人交流。

- 询问功能
  - 例如：`你有什么功能`
- 联网问题
  - 例如：`今天北京天气怎么样`
- 闲聊
  - 例如：`你好`
- 吐槽机器人
  - 例如：`你怎么这么慢`

## Dota2 功能说明

机器人会监控已配置账号的最新比赛，并在发现新比赛后把结果推送到对应群里。

推送内容通常包括：

- 英雄
- 胜负
- KDA
- 比赛时长
- 部分战绩细节
- 一段群聊风格的简短点评

同一个账号可以推送到多个群；`/push <昵称>` 则只会回复到当前群。

## 适合的使用方式

- 在群里查看某个账号的最新一场比赛
- 让机器人自动播报新比赛结果
- 问一些简单的实时问题
- 用 `/todo` 留下新功能需求
